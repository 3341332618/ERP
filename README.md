# ERP管理平台

基于 Spring Boot 3 + Vue 3 的进销存 ERP 管理系统。当前版本实现核心业务闭环：基础资料、采购、销售、库存审核、库存调拨、库存分布、收入结算、支出结算。

## 技术栈

- 后端：Spring Boot 3、Spring Security、JWT、Java 17
- 前端：Vue 3、Vite、TypeScript、Pinia、Vue Router、Element Plus
- 本地数据：后端内置演示数据，启动后可直接登录体验

## 启动方式

后端：

```powershell
cd backend
.\mvnw.cmd spring-boot:run
```

前端：

```powershell
cd frontend
npm.cmd install
npm.cmd run dev
```

访问地址：

```text
http://127.0.0.1:5173
```

## 已实现模块

- 登录、个人中心、系统消息
- 基础信息管理：商品品牌、商品分类、商品单位、商品管理、仓库信息、客户信息、供应商信息
- 采购管理：采购入库、采购退货
- 销售管理：销售出库、销售退货
- 库存管理：库存分布、入库审核、出库审核、库存调拨
- 结算管理：收入结算、支出结算

## 业务规则

- 单据状态使用：待提交、待审核、审核通过、审核拒绝。
- 库存调拨状态在界面中按调拨语义展示：未调拨、待调拨、已调拨、无法调拨。
- 实际库存只在审核通过后变化。
- 可用库存会扣减待提交、待审核、审核拒绝状态的出库类单据数量。
- 采购入库审核通过生成支出结算。
- 采购退货审核通过生成收入结算。
- 销售出库审核通过生成收入结算。
- 销售退货审核通过生成支出结算。
