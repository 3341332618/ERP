## 正在讨论的新方向

用户想在 ERP 后台增加“测试竞赛/缺陷发布”能力，用于实验室学员模拟真实软件测试竞赛。

初步方向：

- 管理员在后台发布可控缺陷。
- 学员登录后执行测试任务并提交缺陷报告。
- 系统记录发现时间、复现步骤、截图/附件、严重级别、状态。
- 教师或管理员审核报告并给分。
- 提供排行榜、任务统计、缺陷覆盖率。
- 缺陷注入应可控，避免永久破坏 ERP 基线功能。
