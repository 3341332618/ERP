# 测试竞赛资料目录

这个目录用于放置测试竞赛相关资料。

建议把缺陷清单放在这里，例如：

```text
docs/requirements/training-lab/bug清单.xlsx
```

后续“缺陷发布/学员测试竞赛”功能会优先读取或参考这个目录中的资料。

```text

```

```text

```
